# 计划 v1.0
移植 applem2 引擎作为 youthm2 引擎，其中 applem2 是 Delphi 项目，它使用的是 Object Pascal 语言，而 youthm2 引擎准备使用 Java 语言开发。

其实可以先略过这部分内容，就像基于接口开发一样，把骨架搭建起来，以后再装修。
